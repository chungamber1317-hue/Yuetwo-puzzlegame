# Yuetwo-puzzlegame
# 悦和酱园拼图解谜游戏

一款基于网页的互动拼图解谜游戏，玩家需要在60秒内完成3x2网格的拼图挑战，拼出悦和酱园的产品图像。
## 如何打開游戲

### 方法：直接打開
1. 雙擊打開 `index..html` 文件
2. 游戲會在默認瀏覽器中運行


## 游戏介绍

这是一个具有计时功能的拼图游戏，玩家需要从下方的碎片池中拖动拼图碎片到上方的对应位置，完成整个图像的拼接。游戏特点包括：

- 60秒倒计时挑战
- 3x2网格的拼图布局
- 实时反馈拼图正确性
- 完成挑战后的庆祝效果
- 响应式设计，适配不同设备

## 游戏规则

1. 游戏开始后，计时器将从60秒开始倒计时
2. 从下方碎片池中拖动拼图碎片到上方对应位置
3. 正确放置的碎片会被固定，错误放置的碎片会在2秒后返回池内
4. 在规定时间内完成所有拼图即可获得胜利
